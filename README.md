# Halcyon for Mastodon
A Mastodon web client

<img src="https://halcyon.social/login/assets/images/preview0.png"/>

## Demo
https://halcyon.social/

## Features
- Twitter like UI, familiar interface.
- Able to use on all instances.
- No tracking, No ads.

## Requirement
- Apache
- PHP
- MySQL
